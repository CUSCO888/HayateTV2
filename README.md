HayateTV 2 📺
HayateTV 2 是一款基于 React 和 Capacitor 构建的跨平台电视直播/视频播放应用。它集成了高性能的 HLS 播放能力，并针对 Android 设备进行了深度优化。

🚀 项目亮点
现代技术栈：采用 React 19 + Vite 6 + TypeScript 5，确保极致的开发体验与运行效率。

跨平台支持：利用 Capacitor 6 架构，实现一套代码同时运行在 Web 和 Android 平台。

高性能播放：集成 hls.js 引擎，支持流畅播放高质量直播流。

优雅 UI：基于 Tailwind CSS 4 和 Framer Motion (Motion) 打造，提供平滑的交互动画。

AI 赋能：内置 Google Gemini API (@google/genai) 接口，预留智能交互扩展空间。

🛠️ 技术架构
前端 (Frontend)
框架: React 19

构建工具: Vite 6

样式: Tailwind CSS 4

动画: Motion (Framer Motion)

状态管理/工具: Lucide React (图标), clsx, tailwind-merge

原生接入 (Native)
容器: Capacitor 6

平台: Android (支持 Java 21 & Gradle 8.11+)

📦 环境要求
在开始构建之前，请确保你的开发环境满足以下条件：

Node.js: v20+

Java SDK: v21 (构建 Android 必选)

Android Studio: 最新稳定版
